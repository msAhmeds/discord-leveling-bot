# 🤖 Discord Economy & Level Bot

A feature-rich Discord bot with an economy system, XP leveling, mini-games, and a virtual shop system.

---

## ✨ Features

- 💬 XP & Leveling system based on messages
- 💰 Virtual economy (coins system)
- 🎰 Mini-games (gambling, dice, etc.)
- 🛒 Shop system for buying items
- 🏆 Leaderboard for top users

---

## ⚙️ Technologies Used

- Python 🐍
- Discord.py
- SQLite database
