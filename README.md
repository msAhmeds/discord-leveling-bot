# 🤖 Discord Economy & Level Bot

A feature-rich Discord bot with an economy system, XP leveling, and mini-games built using Python.

---

## ✨ Features

- 💬 XP & Level system (based on messages)
- 💰 Economy system with coins
- 🎰 Gambling system
- 🎁 Daily rewards
- 📊 Persistent user data (SQLite database)

---

## ⚙️ Technologies Used

- Python 🐍
- Discord.py
- SQLite (database)
